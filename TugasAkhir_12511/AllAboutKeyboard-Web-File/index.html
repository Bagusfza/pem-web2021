Keyboard

Merupakan suatu perangkat keras pada komputer yang berbentuk papan dan memiliki berbagai macam tombol yang dimana tombol tersebut fungsinya berbeda tergantung pada penekanannya yang bisa menghasilkan proses yang sesuai dengan keinginan penggunanya.

Tipe-Tipe Keyboard

Ada 2 macam keyboard yaitu mechanical dan membrane. Jika kalian pernah membongkar keyboard standard yang sudah rusak, kalian akan menemukan satu lempengan karet yang biasa disebut dengan membrane. Karena itu keyboard tersebut biasanya disebut dengan Keyboard Membrane.

Nah karena penggunaan bahan pada keyboard membrane ini merupakan karet yang menyatu, tak jarang jika salah satu bagian karet robek karena pemakaian, akan berpengaruh ke semua bagian keyboard dan daya tahan keyboard ini biasanya hanya 1 tahun saja.
Nah sedangkan untuk keyboard mechanical, jika kita mencabut salah satu key / tuts nya maka akan terlihat bagian yang terlihat independent atau tidak menyatu seperti pada keyboard membrane. Disinilah perbedaan keyboard mechanical dengan membrane paling terlihat. Jika pada keyboard membrane terjadi kerusakan pada bagian karet, maka kita mau tidak mau harus mengganti keyboard baru. Sedangkan pada keyboard mechanical kita hanya perlu mengganti beberapa partnya saja.

Jenis Layout Keyboard

Keyboard memiliki berbagai jenis layout, pada umumnya ada 5 jenis layout sebagai berikut:

Keyboard Layout 40%


Adalah keyboard ketik dengan ukuran paling kecil yang ada di pasaran saat ini. Memang ada beberapa produsen yang membuat keyboard dengan ukuran yang lebih kecil lagi, namun itu tidak digolongkan sebagai keyboard ketik.Keyboard 40 persen ini hanya menyisakan tombol semua huruf dengan ditambah beberapa tombol pendukung, seperti Backspace, Control, Enter, dan sebagainya.

Keyboard Layout 60%


Keyboard ini sedikit lebih besar daripada keyboard 40 persen. Dengan mengutamakan desain yang ringkas, keyboard ini menghilangkan baris fungsi atas (termasuk tombol Escape), tombol panah dan beberapa tombol lain. layout ini sangat digemari di kalangan penggemar keyboard mekanikal yang membutuhkan keyboard yang ringkas ukurannya, tetapi tetap mempunyai tombol fungsional yang lengkap. Keyboard 60 persen ini memiliki 61 tombol untuk standar Amerika (ANSI) atau 62 tombol untuk standar internasional (ISO).

Keyboard Layout 70%


Lebih besar lagi dari keyboard 60 persen, ada keyboard 70 persen. Keyboard in sebenarnya adalah peringkasan dari keyboard TKL. Pengurangan ukuran dari TKL itu dilakukan dengan menghilangkan beberapa tombol seperti tombol panah, tombol naik dan turun, hingga tombol PS/PB/PL.

Keyboard Layout 80% atau TKL


Keyboard ini Disebut Ten Key Less karena keyboard ini menghilangkan semua tombol numerik yang berjumlah sepuluh. Istilah ini kemudian jadi lebih banyak digunakan pada keyboard dengan jumlah tombol 87 buah ini. Sebenarnya, tidak tepat juga disebut Ten Key Less karena jumlah tombol yang dikurangi ada 17 tombol, tapi karena mengacu pada jumlah tombol numeriknya saja yang dihilangkan, maka disebutlah keyboard ini TKL.

Keyboard Layout Full Size


Keyboard ini memiliki 104 tombol yang digunakan untuk semua fungsi mengetik, Keyboard ini juga dapat digunakan untuk semua jenis game yang ada saat ini. Ukurannnya yang terlalu lebar membuat para produsen keyboard mulai menyunat satu persatu tombol yang ada di dalamnya sehingga lahirnya keyboard-keyboard mini seperti di atas.
